  ## 🚀 Proyectos destacados

  ### 🎰 Sistema de Gestión de Eventos y Sorteos
  API REST + WebSocket para gestión de eventos en tiempo real, venta de tickets y sorteos online.

  **Stack:** `NestJS v10` `Prisma ORM` `PostgreSQL 15` `Redis 7` `Socket.io` `Bull` `MercadoPago` `Docker`

  - Autenticación JWT dual (Admin / Usuario) con secretos independientes y RBAC
  - Colas asíncronas con Bull — reservas con TTL automático de 15 min para eliminar conflictos de stock
  - Panel de administración con control de eventos, usuarios y resultados en tiempo real
  - API documentada con Swagger

  ---

  ### 🛒 Ecommerce Backend — API REST
  API REST modular para tienda online con integración de pago, almacenamiento en la nube y locking distribuido.

  **Stack:** `NestJS v11` `Prisma v6` `PostgreSQL 16` `Redis 7` `BullMQ` `MercadoPago` `AWS S3 + CloudFront` `Redlock` `Docker`

  - Integración con MercadoPago para pagos y notificaciones webhook
  - Distributed locking con Redlock para operaciones críticas de inventario — cero condiciones de carrera
  - Autenticación por cookie httpOnly + Bearer fallback con Helmet y CSP configurado
  - Upload, redimensionado y distribución de imágenes vía AWS S3 + CloudFront
